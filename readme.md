# Emacs Init
## Português (Brasil)
No Emacs usamos sequências de teclas descritas por `C-` e `M-` que são para:
* `C-` - _Ctrl +_
* `M-` - _Alt +_ 
* `S-` - _Shift +_
Então o comando `C-x C-c`, para sair do Emacs, é `Ctrl + x` e depois `Ctrl + c`.

### Aplicando configuração
Após instalar o Emacs, utilize `C-x C-f` e digite o diretório de onde foi instalado o clone desse repositório. Após isso, utilize o comando `C-c C-v t` para aplicar as configurações ao arquivo `init.el`. Feche o Emacs, e abra novamente.
